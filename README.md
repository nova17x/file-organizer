# File Organizer

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

散らかったファイルを自動的に整理するGUIアプリケーション。

## 主な機能

- 拡張子、日付、サイズ、ファイル名パターンで自動分類
- プレビュー機能で安全に整理
- バックアップ作成と元に戻す機能
- 重複ファイル検出
- フォルダ監視による自動整理
- ダーク/ライトテーマ切り替え

## インストール

```bash
git clone <repository-url>
cd file-organizer
pip install -r requirements.txt
```

必要要件: Python 3.8以上

## 使い方

```bash
python main.py
```

1. 整理したいフォルダを選択
2. ルールを設定（オプション）
3. プレビューで確認
4. 整理実行
